About Neon Tyrant

Neon Tyrant is a Windows-first action platformer built with C#, Raylib, and native modules for game physics and score persistence. The game focuses on fast, readable movement, tight arena combat, and compact levels you can replay for cleaner runs.

This page is the install portal for stable MSI releases. Install over an existing version to update.

Built for Windows 10/11. Installer target: ARM & x64, coming to macOS soon. Built by Sachin Shankar.
